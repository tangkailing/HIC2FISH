# HiC2FISH
This repository contains the implementation of HiC2FISH, a conditional diffusion framework for generating single-cell FISH-like chromatin distance matrices from bulk Hi-C contact maps.
The full source code, preprocessing scripts, model training scripts, and example inference pipeline will be released upon publication.
For peer review, code can be made available upon reasonable request.
